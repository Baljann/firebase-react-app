# 🎬 Firebase React App

A sleek and minimal **React + Firebase** application for managing your favorite movies. This app uses **Firestore** for real-time data and supports full **CRUD operations** along with **Google Authentication**.

---

## ✨ Features

- 🔐 User authentication (via Firebase + Google Sign-In)
- ➕ Add movies with title, release year, and Oscar status
- 📝 Update movie titles
- ❌ Delete movies from the list
- 🔄 Real-time Firestore integration
